# 考试通考试管理系统

## 1.模块介绍

> -- kst-auth					统一认证模块
>
> -- kst-common			公共类模块
>
> -- kst-gatway				网关模块
>
> -- kst-openFeign		  服务调用
>
> -- kst-service-admin	老师和管理员端业务逻辑
>
> -- kst-student				学生端业务逻辑
>
> -- sql								数据库

## 2.技术栈

### 2.1 运行环境

| 环境     | 版本            |
| -------- | --------------- |
| 操作系统 | Windows / Linux |
| NodeJs   | 14              |
| Jdk      | 1.8             |
| Mysql    | 8.0             |

### 2.2后端系统

- SpringBoot 2.2.2.RELEASE
- SpringCloud Hoxton.SR1
- spring-cloud-alibaba 2.1.0.RELEASE
- mysql 8.0.20
- mybatis-plus 数据库中间件
- nacos 注册中心
- openFeign 服务调用
- gatway 网关，拦截器，验证token
